React Universal

Starter kit for cross-device React app development featuring social login

WHAT IS IT?

React Universal is a boilerplate that helps you get started with developing app that features all three versions of web, mobile, and desktop quick and easy. In addition, React Universal features social login and app-syncing, which are designed to create cross-device, real-time-update apps like you have seen from Slack, Spotify, Skype and many more!

Todos

React Universal Demo

Todos is a live demo of what React Universal is capable of doing. You may see the convinience of utilizing credentials from social logins to store user-related data and the benefits of push events to sync app state across all devices.
